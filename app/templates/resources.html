Databases

  • Kaggle
    •
    • Kaggle is a tremendous resource for collected datasets and competitions in a variety of sports competitions
  • RBSDM
    •
    • Football statistics page run by Ben Baldwin
    • Has many useful datasets as well as beginner nflfastR tutorials
  • nflfastR
    •
    • Website for R package, nflfastR, which has many useful functions and datasets for analyzing the NFL
  • cfbscrapR
    •
    • Github for R package, cfbscrapR, which helps analysis of college football data
  • nbastatR
    •
    • Website for R package, nbastatR, which assists in scraping data and generating analysis for NBA
  • baseballR
    •
    • Github for R package, baseballR, that assists in analyzing MLB analytics
  • Kenpom
    •
    • (Awesome) College MBB analytics website run by Ken Pomeroy
    • Team specific data requires a paid subscription
  • KenpomR
    •
    • R package to scrape Kenpom (team specific data requires subscription)
  • PFF
    •
    • Cutting edge NFL and CFB analytics website run by Cris Collinsworth
    • Most data behind paywall
    • Lots of pretty incredible articles and analysis
  • Basketball Reference
    •
    • Most comprehensive data source for NBA
  • Sports Reference
    •
    • Most comprehensive data source for MBB
  • Football Reference
    •
    • Most comprehensive data source for NFL
  • FB Reference
    •
    • Most comprehensive data source for professional Soccer
  • CFB Reference
    •
    • Most comprehensive data source for college football
  • MLB Reference
    •
    • Most comprehensive data source for MLB
  • Hockey Reference
    •
    • Most comprehensive data source for NHL
  • Sports Data Resources
    •
    • General list of databases for a variety of sports

Learning

  • How to Use R for Sports
    •
    • Provides incredibly basic walkthrough of how to use R to load, clean, and run analysis of sports data
  • Syracuse Analytics Blitz
    •
    • Github of project finished by Jack Lichtenstein, Ben Thorpe, and Bryce Grove for Syracuse Analytics Blitz in February 2021, with the goal of estimating optimal run/pass ratios by field position in the NFL. Excellent example of stunning dataviz, as well as what it looks like to work a project through to completion.
